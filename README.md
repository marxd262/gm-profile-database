# Gaggimate Profile Store

Community catalog for Gaggimate espresso brewing profiles.

## Profiles

This repository contains pressure and flow profiling configurations for the [Gaggimate](https://github.com/marxd262/gaggimate) upgrade for Gaggia espresso machines.

## Usage

Browse profiles at [https://marxd262.github.io/gm-profile-database](https://marxd262.github.io/gm-profile-database)
